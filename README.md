# aicodeanalyser README
AiCodeAnalyser analyses your code, review it, summarize it and describe its operations.

How to Use:
-----------

1. Select any code in the text editor.
2. Right Click.
3. Click "AiCodeAnalyser - Summarise The Code"
4. In some seconds you can see the summary of you code above the selected code block in the form of comments.
